# Civic Bot

A Slack bot,
connected to the Code for America API.

The bot currently supports the following commands:

`dot`
`dot cfa`
`dot cfa:organizations`
`dot cfa:projects`
`dot cfa:events`
`dot cfa:stories`
`dot cfa:issues`
